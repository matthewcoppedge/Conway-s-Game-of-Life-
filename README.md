# Conway's Game of Life
Implementation of Conway's Game of Life w/SFML &amp; C++

## SFML Installation
**Linux**: sudo apt-get install libsfml-dev

## Compile
g++ main.cpp -lsfml-system -lsfml-window -lsfml-graphics -g -O3
